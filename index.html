Bootparts

A collection of Bootstrap components that make building websites much easier.

Documentation Download

Current Release: v0.3.0

Desgined for people with taste and deadlines.

Features ready-to-use templates that save you time and headaches (and are responsive).

Super Easy To Use.

It was designed to be convenient. You will pretty much copy and paste whatever components you need and then apply some custom styles. Worry about the important stuff. Not HTML.

Oh, and by the way, its built with mobile and responsiveness in mind.

üëç

No need to use Javascript.

Seriously. You don't need it. It includes Jquery (because Bootstrap already does). But it's not used. Not that Javascript is bad. This project tries to keep things simple in order to keep you sane.

üëè

Includes a set of useful templates.

No need to re-invent anything. Most websites are very similar. Adjust a color here, some padding there, and you are done. The templates try to follow a given theme, but you are not forced to do anything. Best of all? The templates are released under the MIT license as well. Yes, they are Free (as in beer). More templates are planned.

Bootparts is Open Source and released under the MIT license (same as Bootstrap).
